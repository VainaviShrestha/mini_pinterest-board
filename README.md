# Mini-Pinterest

A simple Mini-Pinterest app built with HTML, CSS, and JavaScript.

---

## Features

- Add image cards with a title  
- Like/favorite cards  
- Delete cards  
- Drag-and-drop to rearrange  
- Cards stay after refresh (localStorage)

---

## How to Run

1. Open the project folder.  
2. Open `index.html` in your browser.  
3. (Optional) Use Live Server in VS Code for auto-refresh.
